# Chess UTF-8

Chess game using UTF-8 characters.

## Install

1. Clone this repository:

`$ git clone https://github.com/jazielloureiro/Chess-UTF-8`

2. Go into the respository directory:

`$ cd Chess-UTF-8`

3. And then, run the Makefile:

`$ make chess`
